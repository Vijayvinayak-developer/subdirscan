## 📁 Directory & Subdomain Bruteforce Tool

This project is an automated Python-based tool designed for **directory** and **subdomain bruteforcing**.  
It helps in discovering hidden directories and subdomains using wordlists, making it useful for reconnaissance, learning, and controlled security testing.

---

## ⚙️ Features

- 🔍 Directory bruteforcing  
- 🌐 Subdomain enumeration  
- 📄 Supports custom wordlists  
- 📦 Includes default wordlists  
- 🖥️ Easy execution via script files (Windows & Linux)

---

## 🚀 Getting Started

### ▶️ Run on Windows

Double-click on:

```
runner.bat
```

---

### ▶️ Run on Linux

Give execution permission:

```
chmod +x runner.sh
```

Run the script:

```
./runner.sh
```

---

## 📌 Notes

- No additional parameters are required to run the script  
- Make sure Python is installed on your system  
- Customize wordlists for better results
